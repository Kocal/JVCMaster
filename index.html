JVCMaster

JVCMaster est une extension navigateur compatible avec Google Chrome 21+, Chromium 18+, Mozilla Firefox 14+, Opera 12+, ajoutant des fonctionnalités aux forums de Jeuxvideo.com.

Via un panneau de gestion d'extensions, activez ou non des fonctionnalités d'un simple clic.



Maintenant, vous pouvez :

  • Afficher des informations d'un pseudo à côté de celui-ci (sexe, rang)
  • Afficher les images de NoelShack dans une lightbox
  • Cacher les posts des bots (flood)
  • Citer un ou des post(s)
  • Cacher un ou des posts, et les posts d'un pseudo

Télécharger l'UserScript Voir sur GitHub

Installer JVCMaster

Google Chrome
  • Cliquez sur le bouton Télécharger l'UserScript
  • Ouvrez la page d'extensions
  • Ouvrez votre dossier de téléchargements
  • Glissez JVCMaster.user.js sur la page d'extensions de Google Chrome

Chromium
  • Cliquez sur le bouton Télécharger l'UserScript

Mozilla Firefox
  • Téléchargez GreaseMonkey
  • Cliquez sur le bouton Télécharger l'UserScript

Opera
  • Cliquez sur le bouton Télécharger l'UserScript
  • Si votre dossier JavaScript n'est pas configuré
    • Créez un dossier qui contiendra les scripts JavaScript pour Opera
    • Déplacez JVCMaster.user.js dans ce dossier
    • Cliquez sur le O (menu) > Réglages > Préférences > Avancé > Contenu > Options JavaScript
    • Cliquez sur le bouton "Choisir...", et choisissez le dossier que vous venez de créer
  • Si votre dossier JavaScript est déjà configuré
    • Déplacez JVCMaster.user.js dans ce dossier
